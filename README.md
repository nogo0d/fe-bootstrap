## fe-bootstrap


run development mode
 > yarn start

 create optimized production build

 > yarn build

... run it:
  > yarn build:run



 run tests

 > yarn test



 check repository code integrity

 > yarn cicheck



 check code linting rules

 > yarn lint
 
 ... fix it:

 > yarn lint:fix


play with demo components

 > yarn storybook

publish build in registry

 > yarn publish
